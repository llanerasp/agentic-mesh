# Lessons — Aprendizajes y correcciones

Este archivo se va alimentando a medida que avanzamos. Cada vez que:

- El usuario me corrija algo no obvio.
- Me tope con un error que me costó diagnosticar.
- Descubra que una decisión del roadmap era incorrecta.
- Aprenda algo que cambie mi forma de enfocar problemas similares.

…lo anoto aquí con el formato:

```
## [Fecha] — Título corto

**Qué pasó**: descripción del problema o corrección.
**Por qué**: la causa raíz o el razonamiento que faltaba.
**Regla para futuro**: cómo evitar repetirlo.
```
